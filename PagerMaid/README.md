

# PagerMaid 安装指南



## ⚙️ 运行

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/EAlyce/conf/main/PagerMaid/RXsetup.sh)
````
所需其他参数

`api_id`  
`api_hash`

申链链接：[点击跳转](https://my.telegram.org/auth)  填写表单后即可获取

`number` 你的电话号码 格式：+区号电话号码 例如：+8618888888888

`password` 你的二次验证密码（如果有）
