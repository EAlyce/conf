# 插件列表

| 插件名       | 简要说明                     | 安装命令              |
|--------------|------------------------------|-----------------------|
| **sb**       | 跨群封禁用户并可删除其消息   | `,apt install sb`      |
| **shift**    | 智能频道转发助手             | `,apt install shift`   |
| **unban_self**| 一键解封自己封禁的用户       | `,apt install unban_self` |
| **bf**       | 备份历史消息                 | `,apt install bf`      |
